# Description
An e-commerce website selling second-hand vinyl records.
Featuring the full CRUD functionality made with Node.js.
Uploading images enabled through Cloudinary.
App hosted on Heroku.

# Technologies Used
- JavaScript
- Node.js
- Express
- MongoDB
- Mongoose
- CSS
- Bootstrap
- HTML
- Cloudinary
- Multer
